# ABC Wonder — Product Vision

## Elevator Pitch
A colorful, zero-pressure kids' learning app for alphabet, numbers, days, and months. Designed for 2–6 year olds. No ads, no streaks, no addiction loops.

## Target Audience
- **Primary**: Children aged 2–6
- **Secondary**: Parents who want safe, educational screen time

## Core Philosophy
> "The app is a tool, not a babysitter."

- **Tap-driven** — no autoplay, no auto-advance
- **No rewards / coins / points** — learning is the reward
- **No notifications** — the app never pulls the child back
- **No data collection** — 100% offline, no internet required
- **No ads or in-app purchases** — free forever

## Features (MVP — Ship in 5 Days)

| Feature | Priority | Effort |
|---|---|---|
| Alphabet A–Z with TTS audio | P0 | 1 day |
| Numbers 1–100 with object counting | P0 | 1 day |
| Days of the week | P0 | 0.5 day |
| Months of the year | P0 | 0.5 day |
| Responsive phone ↔ tablet | P0 | 1 day |
| Colorful kid-friendly UI | P0 | 1 day |
| Smooth transitions | P1 | 0.5 day |

## Non-Goals (v1)
- Games / quizzes
- Writing / tracing
- Parent dashboard
- Analytics or tracking
- Multiple languages
- Animations beyond basic transitions

## Success Criteria
1. App installs and opens on Android & iOS
2. All 4 categories render correctly on phone (360dp) and tablet (600dp+)
3. Tapping any letter/number/day/month shows its name + plays TTS audio
4. No crashes in 10-minute kid-driven session
5. APK size < 30 MB

## Monetization
None. Free forever. Built as a learning tool.
